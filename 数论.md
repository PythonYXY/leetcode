# 202. 快乐数
```
编写一个算法来判断一个数是不是“快乐数”。

一个“快乐数”定义为：对于一个正整数，每一次将该数替换为它每个位置上的数字的平方和，然后重复这个过程直到这个数变为 1，也可能是无限循环但始终变不到 1。如果可以变为 1，那么这个数就是快乐数。

示例: 

输入: 19
输出: true
解释: 
12 + 92 = 82
82 + 22 = 68
62 + 82 = 100
12 + 02 + 02 = 1

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/happy-number
著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。
```

## 代码
```java
class Solution {
    public boolean isHappy(int n) {
        int slow = n, fast = n;
        
        // 用快慢指针找到循环的入口
        while (true) {
            slow = next(slow);
            fast = next(next(fast));
            if (slow == fast) return slow == 1;
        }
    }

    int next(int n) {
        int sum = 0;
        while (n != 0) {
            int num = n % 10;
            sum += num * num;
            n /= 10;
        }

        return sum;
    }
}
```

```java
class Solution {
    public int nthUglyNumber(int n) {
        PriorityQueue<Long> pq = new PriorityQueue<>();
        pq.add(Long.valueOf(1));
        long top = -1;
        for (int i = 0; i < n; i++) {
            top = pq.poll();
            while (!pq.isEmpty() && pq.peek() == top) pq.poll();
            pq.add(2 * top);
            pq.add(3 * top);
            pq.add(5 * top);
        }
        
        return (int)top;
    }
}
```

```java
class Solution {
    public int nthUglyNumber(int n) {
        int[] num = new int[n];
        num[0] = 1;

        int i = 0, j = 0, k = 0, index = 0;
        while (index < n - 1) {
            int minNum = Math.min(num[i] * 2, Math.min(num[j] * 3, num[k] * 5));
            if (minNum == num[i] * 2) i++;
            if (minNum == num[j] * 3) j++;
            if (minNum == num[k] * 5) k++;
            if (minNum != num[index]) num[++index] = minNum;
        }

        return num[n - 1];
    }
}
```

# 313. 超级丑数
```
编写一段程序来查找第 n 个超级丑数。

超级丑数是指其所有质因数都是长度为 k 的质数列表 primes 中的正整数。

示例:

输入: n = 12, primes = [2,7,13,19]
输出: 32 
解释: 给定长度为 4 的质数列表 primes = [2,7,13,19]，前 12 个超级丑数序列为：[1,2,4,7,8,13,14,16,19,26,28,32] 。
说明:

1 是任何给定 primes 的超级丑数。
 给定 primes 中的数字以升序排列。
0 < k ≤ 100, 0 < n ≤ 106, 0 < primes[i] < 1000 。
第 n 个超级丑数确保在 32 位有符整数范围内。

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/super-ugly-number
著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。
```

```java
class Solution {
    public int nthSuperUglyNumber(int n, int[] primes) {
        int[] num = new int[n + 1];
        num[0] = 1;
        int[] pIndex = new int[primes.length];
        int index = 0;
        for (int i = 0; i < n; i++) {
            int min = Integer.MAX_VALUE;
            for (int j = 0; j < primes.length; j++) min = Math.min(min, num[pIndex[j]] * primes[j]);
            for (int j = 0; j < primes.length; j++) {
                if (min == num[pIndex[j]] * primes[j]) {
                    pIndex[j]++;
                }
            }

            if (min != num[index]) num[++index] = min;
        }

        return num[n - 1];

    }
}
```

```java
class Solution {

    public String fractionToDecimal(int a, int b) {
        long numerator = Long.valueOf(a);
        long denominator = Long.valueOf(b);

        if (numerator == 0) return "0";
        StringBuilder res = new StringBuilder();
        if (numerator < 0 && denominator > 0 || numerator > 0 && denominator < 0) {
            numerator = Math.abs(numerator);
            denominator = Math.abs(denominator);
            res.append("-");
        }
        res.append(String.valueOf(numerator / denominator));
        numerator %= denominator;

        if (numerator == 0) {
            return res.toString();
        } else {
            res.append(".");
            
            Map<Long, Integer> map = new HashMap<>();
            map.put(numerator, res.length());

            while (numerator != 0) {

                long reminder = (numerator * 10) % denominator;
                if (map.containsKey(reminder)) {
                    res.insert(map.get(reminder), "(");
                    res.append((numerator * 10) / denominator);
                    res.append(")");
                    return res.toString();
                }

                long divisor = (numerator * 10) / denominator;
                res.append(String.valueOf(divisor));

                map.put(reminder, res.length());
                numerator = reminder;
            }
            return res.toString();
        }
    }
}
```

# 172. 阶乘后的零
```
给定一个整数 n，返回 n! 结果尾数中零的数量。

示例 1:

输入: 3
输出: 0
解释: 3! = 6, 尾数中没有零。
示例 2:

输入: 5
输出: 1
解释: 5! = 120, 尾数中有 1 个零.
说明: 你算法的时间复杂度应为 O(log n) 。

来源：力扣（LeetCode）
链接：https://dev.lingkou.xyz/problems/factorial-trailing-zeroes
著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。
```
## 思路
n!中有多少个0，取决于n!中一共有多少个质因数2和5，由于2的个数一定大于5，所以最终取决于有多少个5。

## 代码
```java
// Factorial Trailing Zeroes
// Time Complexity: O(logn), Space Complexity: O(1)
public class Solution {
    public int trailingZeroes(int n) {
        int result = 0;

        while (n > 0) {
            result += n / 5;
            n /= 5;
        }
        return result;
    }
}
```

```
给定一个非空的整数数组，返回其中出现频率前 k 高的元素。

示例 2:

说明：

你可以假设给定的 k 总是合理的，且 1 ≤ k ≤ 数组中不相同的元素的个数。
你的算法的时间复杂度必须优于 O(n log n) , n 是数组的大小。

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/top-k-frequent-elements
著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。
```
## 代码
```java
class Solution {
    public List<Integer> topKFrequent(int[] nums, int k) {
        Map<Integer, Integer> map = new HashMap<>();

        for (int i = 0; i < nums.length; i++) {
            map.put(nums[i], map.getOrDefault(nums[i], 0) + 1);
        }

        PriorityQueue<Map.Entry<Integer, Integer>> pq = new PriorityQueue<>((e1, e2) -> {
            return e2.getValue() - e1.getValue();
        });

        for (Map.Entry<Integer, Integer> entry: map.entrySet()) {
            pq.add(entry);
        }

        List<Integer> res = new ArrayList<>();
        while (k-- > 0) {
            res.add(pq.poll().getKey());
        }
        return res;
    }
}
```

```java
class Twitter {
    class Node {
        int tweetId;
        int time;
        Node next;

        public Node(int tweetId, int time) {
            this.tweetId = tweetId;
            this.time = time;
        }
    }

    private Map<Integer, Node> tweet;
    private Map<Integer, Set<Integer>> followSet;
    private int currentTime;

    /** Initialize your data structure here. */
    public Twitter() {
        tweet = new HashMap<>();
        followSet = new HashMap<>();
        currentTime = 0;
    }
    
    /** Compose a new tweet. */
    public void postTweet(int userId, int tweetId) {
        if (!tweet.containsKey(userId)) tweet.put(userId, new Node(tweetId, currentTime++));  
        else {
            Node node = new Node(tweetId, currentTime++);
            node.next = tweet.get(userId);
            tweet.put(userId, node);
        }
    }
    
    /** Retrieve the 10 most recent tweet ids in the user's news feed. Each item in the news feed must be posted by users who the user followed or by the user herself. Tweets must be ordered from most recent to least recent. */
    public List<Integer> getNewsFeed(int userId) {
        PriorityQueue<Node> pq = new PriorityQueue<>((n1, n2) -> {
            return n2.time - n1.time;
        });

        if (tweet.containsKey(userId)) pq.add(tweet.get(userId));
        
        if (followSet.containsKey(userId)) {
            Iterator<Integer> ite = followSet.get(userId).iterator();
            while (ite.hasNext()) {
                int followeeId = ite.next();
                if (tweet.containsKey(followeeId)) pq.add(tweet.get(followeeId));
            }
        }

        List<Integer> ret = new ArrayList<>();
        while (!pq.isEmpty() && ret.size() < 10) {
            Node node = pq.poll();
            ret.add(node.tweetId);
            if (node.next != null) pq.add(node.next);
        }
        return ret;
    }
    
    /** Follower follows a followee. If the operation is invalid, it should be a no-op. */
    public void follow(int followerId, int followeeId) {
        if (followerId == followeeId) return;
        if (!followSet.containsKey(followerId)) followSet.put(followerId, new HashSet<>());
        followSet.get(followerId).add(followeeId);        
    }
    
    /** Follower unfollows a followee. If the operation is invalid, it should be a no-op. */
    public void unfollow(int followerId, int followeeId) {
        if (followerId == followeeId) return;
        if (followSet.containsKey(followerId) && followSet.get(followerId).contains(followeeId)) followSet.get(followerId).remove(followeeId);
    }
}
```

# 373. 查找和最小的K对数字
```
给定两个以升序排列的整形数组 nums1 和 nums2, 以及一个整数 k。

定义一对值 (u,v)，其中第一个元素来自 nums1，第二个元素来自 nums2。

找到和最小的 k 对数字 (u1,v1), (u2,v2) ... (uk,vk)。

     [1,1],[1,1],[1,2],[2,1],[1,2],[2,2],[1,3],[1,3],[2,3]
示例 3:

输出: [1,3],[2,3]
```

## 代码

```java
class Solution {
    class Node {
        int index1, num1, index2, num2;

        public Node(int index1, int num1, int index2, int num2) {
            this.index1 = index1;
            this.num1 = num1;
            this.index2 = index2;
            this.num2 = num2;
        }
    }

    public List<List<Integer>> kSmallestPairs(int[] nums1, int[] nums2, int k) {
        if (nums1.length == 0 || nums2.length == 0) return new ArrayList<>();

        PriorityQueue<Node> pq = new PriorityQueue<>((n1, n2) -> {
            return n1.num1 + n1.num2 - n2.num1 - n2.num2;
        });
        pq.add(new Node(0, nums1[0], 0, nums2[0]));
        Set<String> set = new HashSet<>();

        int index1 = 0, index2 = 0;
        List<List<Integer>> ret = new ArrayList<>();
        
        while (!pq.isEmpty() && ret.size() < k) {
            Node node = pq.poll();
            ret.add(Arrays.asList(node.num1, node.num2));
            if (node.index1 < nums1.length - 1 && !set.contains(getKey(node.index1 + 1, node.index2))) {
                pq.add(new Node(node.index1 + 1, nums1[node.index1 + 1], node.index2, node.num2));
                set.add(getKey(node.index1 + 1, node.index2));
            }
            if (node.index2 < nums2.length - 1 && !set.contains(getKey(node.index1, node.index2 + 1))) {
                pq.add(new Node(node.index1, node.num1, node.index2 + 1, nums2[node.index2 + 1]));
                set.add(getKey(node.index1, node.index2 + 1));
            }
        }

        return ret;
    }

    public String getKey(int index1, int index2) {
        return String.valueOf(index1) + "-" + String.valueOf(index2);
    }
}
```
